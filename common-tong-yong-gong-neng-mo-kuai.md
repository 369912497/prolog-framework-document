# **common - 通用功能模块**

> #### 模块名称

```
plg-fx-common
```

> #### 引用

```
<dependency>
     <groupId>com.prolog.framework</groupId>
     <artifactId>plg-fx-common</artifactId>
     <version>${plg.fx.verison}</version>
 </dependency>
```

> #### 配置

```
无
```

> #### 使用

* 在controller中使用注解 @ControllerLog 进行注解，可注解在类上，亦可注解在方法上
