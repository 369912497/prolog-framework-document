# service** - web服务模块**

> #### 模块名称

```
plg-fx-service
```

> #### 引用

```
<dependency>
     <groupId>com.prolog.framework</groupId>
     <artifactId>plg-fx-starter-service</artifactId>
     <version>${plg.fx.verison}</version>
 </dependency>
```

> #### 配置

```
prolog:
  web:
    multipart:  #文件上传配置
      maxFileSize: 100MB
      maxRequestSize: 100MB
```

> #### 使用

* 提供Message封装
